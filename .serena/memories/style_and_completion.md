Conventions: main.py is the preferred human-facing entrypoint. Python ingestion/discovery modules write structured CSV/TXT/JSONL/JSON outputs and sync canonical `*_latest` artifacts. Multi-step workflows belong in app/commands/pipeline.py. Root-level scripts are compatibility shims; new implementation code should live under app/. When finishing a task, verify syntax/help output locally, prefer explicit artifact paths in config.py, and update task_plan.md, findings.md, and progress.md for multi-step work.
